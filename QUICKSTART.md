## 快速启动（后端）

### 1. 安装 Node.js
```bash
# Windows: https://nodejs.org/
# Mac: brew install nodejs
# Linux: sudo apt install nodejs
```

### 2. 安装 MySQL
```bash
# Windows: https://dev.mysql.com/downloads/mysql/
# Linux: sudo apt install mysql-server
```

### 3. 克隆并启动
```bash
git clone https://github.com/greasebig/uav-dispatch-platform.git
cd uav-dispatch-platform
pnpm install
pnpm db:push  # 初始化数据库
pnpm dev      # 启动服务
```

访问 http://localhost:3000

## 常见问题

### Q: 支付功能无法使用？
A: 需要去 stripe.com 注册账号，获取API Key

### Q: 地图无法显示？
A: 需要去 Google Maps / 高德地图 申请API Key

### Q: 如何成为管理员？
A: 数据库中手动将用户角色设为 admin

### Q: 鸿蒙系统能用吗？
A: 可以，APK兼容运行

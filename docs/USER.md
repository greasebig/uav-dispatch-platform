# 无人机调度平台用户使用指南

## 一、平台概述

本平台是一款专注于无人机作业调度的移动应用，旨在连接需求用户与专业飞手，实现高效、便捷的无人机作业服务。平台提供两大核心业务板块：喷药作业和吊运作业，满足不同用户的多样化需求。

平台致力于构建一个安全、规范的无人机作业环境。系统内置禁飞区管理功能，用户在特定区域发单时将收到安全提示；同时，平台对所有聊天内容进行合规审核，确保交流环境健康有序。在收费方面，平台采用透明化的信息费收取机制，用户支付信息费后可获取飞手联系方式，作业费用则由用户与飞手私下结算，平台仅收取相应的服务佣金。

## 二、功能模块

### 2.1 用户注册与登录

新用户首次使用本应用时，需要完成账号注册。注册流程设计简洁高效，用户只需填写手机号码、设置登录密码、完成短信验证码验证即可成功注册。已注册用户可通过手机号码和密码登录，亦可选择手机号码加短信验证码的快捷登录方式。

登录成功后，用户将进入应用主界面。主界面采用底部导航栏设计，包含首页、订单、消息和个人中心四大核心板块。用户可根据自身需求快速切换不同功能模块。值得注意的是，为保障账号安全，建议用户定期更换登录密码，并妥善保管好个人账户信息，避免账号被盗用。

### 2.2 实名认证

完成注册后，用户需要进行实名认证才能使用平台的完整功能。实名认证流程采用OCR自动识别技术，用户只需拍摄身份证正反面照片，系统将自动提取身份信息并完成认证。认证过程中，系统会要求用户进行人脸识别验证，以确保操作者为账号所有者本人。

实名认证审核通常在提交后24小时内完成。审核结果将通过短信和应用内通知两种方式告知用户。认证成功后，用户即可解锁平台全部功能，包括发布订单、联系飞手、预约服务等。若认证失败，系统将显示具体失败原因，用户可根据提示重新提交认证材料。

### 2.3 发布订单

用户登录并完成实名认证后，即可开始发布无人机作业订单。发布订单是本平台的核心功能之一，用户可根据实际需求选择喷药作业或吊运作业两种类型。

**喷药作业订单**适用于农林植保场景。用户需要填写以下信息：作业面积（亩）、作业地点（支持地图定位）、作业时间、作物类型、药剂类型、特殊要求等。系统会根据作业面积和当前市场的指导价格，显示预估费用供参考。用户还可以上传作业区域的照片或视频，帮助飞手更准确地评估作业条件。

**吊运作业订单**适用于货物运输、物资吊装等场景。用户需要填写：货物重量（公斤）、起运地点、目的地、作业时间、货物类型、吊运要求等。同样，系统会根据货物重量和运输距离显示预估费用。用户可上传货物照片，以便飞手准备相应的设备和工具。

订单提交后，系统将进行禁飞区检查。若用户选择的作业地点位于禁飞区内，系统会弹出安全提示，提醒用户确认是否继续发布。确认后，订单将正常发布，但飞手在接单前也会看到相应的禁飞区提示。

### 2.4 费用说明

在本平台使用过程中，用户需要了解两项费用的区别：

**信息费**是用户支付给平台的技术服务费用，用于获取飞手的联系方式。用户浏览订单详情时，可看到飞手的基本信息和服务评价，但联系方式需要支付信息费后方可查看。信息费的计算方式如下：喷药作业按实际作业费用的10%收取，每单最低5元；吊运作业按实际作业费用的8%收取，每单最低10元。信息费由用户在确认联系飞手时支付，采用微信支付或支付宝两种方式。

**作业费用**是用户支付给飞手的实际服务报酬，包括设备使用费、飞行费、药剂费等。用户在获取飞手联系方式后，可与飞手私下协商作业费用的具体金额和支付方式。平台仅作为信息撮合方，不参与作业费用的直接交易。作业完成后，用户可根据服务满意度对飞手进行评价。

### 2.5 搜索与匹配

平台提供智能订单匹配功能，帮助用户快速找到合适的飞手。用户可在订单列表中浏览符合条件的飞手信息，也可以使用搜索功能按照地区、评分、服务类型等条件进行筛选。

搜索功能支持多种筛选条件：地区筛选可精确定位到具体城市或区县；评分筛选可选择4.5星以上、4星以上等不同等级；服务类型筛选可选择仅显示喷药飞手、仅显示吊运飞手或两者兼有；距离排序可按照与作业地点的距离由近到远排列。此外，用户还可以查看飞手的详细资料，包括从业年限、服务次数、用户评价、擅长机型等，帮助做出选择。

### 2.6 在线沟通

平台内置即时通讯功能，用户与飞手可通过应用内聊天窗口进行沟通。聊天功能支持文字消息、语音消息、图片发送等多种形式，方便双方就作业细节进行详细沟通。

为保障交流环境的健康合规，平台对聊天内容进行实时安全监测。系统会自动过滤包含违法内容（如赌博、色情、毒品等）和敏感信息（如私下交换电话号码、微信号、QQ号等）的消息。触发违禁词的消息将无法发送，同时系统会向发送方提示"消息包含不合规内容，请注意文明交流"。若用户多次发送违禁消息，系统将对该账号进行警告，情节严重者将限制其聊天功能甚至封禁账号。

建议用户通过平台官方渠道进行沟通和交易，勿轻易相信私下转账或提前支付等行为，以保障自身权益。

### 2.7 订单管理

用户可在"订单"模块查看所有已发布的订单及其状态。订单状态包括：待接单（订单已发布，等待飞手接单）、已接单（已有飞手接单，等待作业）、进行中（飞手正在作业）、已完成（作业已完成，用户已确认）、已取消（订单被用户或平台取消）。

对于已发布的订单，用户可进行以下操作：查看订单详情、取消订单（待接单状态下可取消）、联系飞手（需支付信息费）、查看作业进度、对已完成订单进行评价。若在作业过程中出现纠纷，用户可在订单详情页发起投诉，平台客服将在24小时内介入处理。

### 2.8 投诉与售后

平台提供完善的投诉与售后服务体系。用户在使用过程中遇到任何问题，均可通过以下渠道寻求帮助：

**应用内投诉**：在订单详情页点击"投诉"按钮，选择投诉类型（服务质量、费用纠纷、安全问题、其他），填写投诉详情并提交相关证据。平台客服将在24小时内响应处理。

**在线客服**：点击个人中心页的"在线客服"按钮，即可接入人工客服。客服工作时间为每日9:00至21:00，法定节假日正常值班。

**客服热线**：用户也可拨打平台客服热线进行咨询和投诉。热线电话在个人中心页面显示，请留意查看。

### 2.9 个人中心

个人中心是用户管理个人信息和账户资产的入口。主要功能包括：

**账户信息**：查看和修改个人资料，包括头像、昵称、联系电话等。实名认证状态和认证信息也在此展示。

**我的订单**：查看所有订单记录，包括发布订单和接受订单（飞手角色）。

**我的收藏**：收藏感兴趣的飞手或常用地址，方便下次快速查找。

**地址管理**：管理常用作业地址，发布订单时可快速选择。

**支付记录**：查看信息费支付记录和发票开具情况。

**设置**：包括通知设置、隐私设置、账号安全等选项。

## 三、安全须知

### 3.1 禁飞区提示

本平台已集成全国主要机场、军事禁区等禁飞区域数据。当用户发布的订单作业地点位于禁飞区内时，系统将自动弹出安全提示窗口，告知用户该区域为禁飞区域，并要求用户确认是否继续发布。

根据民航管理规定，无人机在禁飞区内飞行需要获得相应审批。用户选择继续发布订单即表示已知悉禁飞区相关规定，并承诺在作业前办理相关飞行审批手续。未办理审批在禁飞区飞行属于违法行为，将承担相应法律责任。平台提醒广大用户务必遵守法律法规，安全合法使用无人机。

### 3.2 隐私保护

平台高度重视用户隐私保护。在聊天过程中，请勿轻易泄露个人银行卡号、密码、验证码等金融信息，平台工作人员不会以任何理由向用户索要此类信息。如遇可疑情况，请及时联系平台客服核实。

用户与飞手线下交易时，应选择安全场所并注意人身财产安全。建议在公共场所进行现金交易或转账操作，避免遭受损失。

### 3.3 账号安全

请妥善保管登录密码和支付密码，建议使用复杂密码并定期更换。不要将账号借给他人使用，以免造成个人信息泄露或财产损失。如发现账号异常登录情况，请及时通过"忘记密码"功能重置密码，并检查账户是否有未经授权的操作。

## 四、常见问题

**问：如何联系飞手获取联系方式？**

答：在订单详情页面点击"联系飞手"按钮，系统将引导完成信息费支付。支付成功后即可查看飞手的联系电话和微信号码。信息费支持微信支付和支付宝支付。

**问：信息费可以退还吗？**

答：信息费支付后24小时内可申请退款，超过24小时或已与飞手见面沟通后不支持退款。如遇特殊情况，请联系平台客服处理。

**问：订单发在禁飞区会有问题吗？**

答：系统会提示禁飞区信息，用户确认后订单可正常发布。但禁飞区飞行需要提前办理审批手续，建议用户了解相关法规后再进行作业。

**问：与飞手产生纠纷怎么办？**

答：可在订单详情页发起投诉，提交相关证据。平台客服将进行调查处理，并根据情况采取扣除飞手保证金、限制接单、永久封号等措施。

**问：如何成为飞手提供服务？**

答：打开个人中心，点击"我要成为飞手"，提交飞行执照、设备证明、身份证明等材料。平台审核通过后即可接单。
